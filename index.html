Development Projects

  1. LIBSIB : A C++ library for computing Smallest Intersecting Balls for point sets, convex polytopes, reduced polytopes, AABBs, balls and ellipsoids in arbitrary dimensions.

  2. PDSCP (to be released) : A library for computing Smallest Enclosing Balls and Support Vector Machines in high-dimensional spaces using GPU. It is a practical demonstration of our primal-dual algorithmic framework for solving Symmetric Cone Programs.

  3. PosLP (to be released) : A library for solving Positive Linear Programs using GPU, which implements and improves the linear coupling algorithm proposed by Allen-Zhu and Orecchia.

  4. PBA+ ( 73 | 8 ) : The most efficient open-sourced library for computing 2D and 3D Digital Voronoi Diagrams using GPU. It can process very large input images (up to 32K x 32K pixels) within a second. The source code has been integrated into game engines and motion-planning projects.

  5. gCVT ( 35 | 10 ) : An open-sourced library for computing 2D and 3D Centroidal Voronoi Tessellations (CVTs) using GPU. It adopts the over-relaxed Lloyd's method for minimizing the objective function and outperforms existing optimizers.

  6. SurfaceRemesher ( 37 | 6 ) : An open-sourced project for optimizing Surface Meshes, which parameterizes triangulated surface meshes in planar spaces and optimizes the positions of the vertices via CVTs.
